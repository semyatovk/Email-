
1. Regex дегеніміз не?
Regex (Regular Expression) — мәтіндегі белгілі бір үлгіні (pattern) табу және тексеру үшін қолданылатын арнайы ережелер жиынтығы.
Ол деректердің дұрыс форматта екенін тексеруге көмектеседі (email, телефон, пароль).
2. Email қандай шарттарға сәйкес болуы керек?
Дұрыс email үшін негізгі талаптар:
@ символы болуы керек
@ алдында пайдаланушы аты (name) болуы керек
@ кейін домен болуы керек (mail, gmail, т.б.)
Соңында домен аймағы болуы керек (.com, .kz, .ru)
Бос орын болмауы керек
3. String тексеру қалай жүзеге асады?
String тексеру екі әдіспен жасалады:
✔ Regex арқылы
Белгілі бір үлгі (pattern) құрылады
Сол үлгіге сәйкес келуі тексеріледі
✔ Қарапайым әдіс
@ бар ма → in немесе contains()
Ұзындығы → len()
Нүкте бар ма → "." in string
Бірақ ең тиімді және нақты әдіс — Regex.
